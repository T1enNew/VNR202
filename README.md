# Cương lĩnh xây dựng đất nước trong thời kỳ quá độ lên CNXH

Bài thuyết trình web dạng **quyển sách lật trang 3D** cho học phần *Lịch sử Đảng Cộng sản Việt Nam — Chương 4* (giai đoạn 1991 đến nay).

## Nội dung
- **Phần 1 — Cương lĩnh 1991:** bối cảnh ra đời, 6 đặc trưng của xã hội XHCN.
- **Phần 2 — Đại hội X (2006):** điểm bổ sung, 3 nhiệm vụ then chốt, kết quả hội nhập.
- **Phần 3 — Cương lĩnh 2011 (trọng tâm):** 5 bài học, 8 phương hướng, 8 mối quan hệ lớn.
- **Phần 4 — Liên hệ thực tiễn:** kinh tế, xây dựng Đảng, đối ngoại, khát vọng 2025–2030–2045.

## Cách dùng
Mở `index.html` bằng trình duyệt. Điều khiển:
- **→ / ←** (hoặc phím cách): lật trang tới / lui
- Bấm **mép trái / phải** của sách để chuyển trang
- **F**: bật/tắt toàn màn hình

## Chạy online
File là một trang tĩnh duy nhất — có thể bật **GitHub Pages** (Settings → Pages → nhánh `main`) để có link trình chiếu trực tiếp.
